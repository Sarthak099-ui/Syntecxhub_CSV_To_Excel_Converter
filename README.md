# CSV to Excel Converter (SyntecXHub Internship Task)

A modular, robust command-line Python utility that automates data preparation by ingestion, cleaning, normalizing, and converting raw CSV files into structured, production-ready Excel (`.xlsx`) files.

## 🚀 Features
- **Data Normalization:** Automatically transforms column headers to standard lowercase `snake_case`.
- **Intelligent Cleaning:** Trims accidental white spaces, replaces blank text fields with `N/A`, and fills empty numerical data with `0`.
- **Comprehensive Logging:** Tracks execution workflow and details errors inside a local `converter.log` file.
- **Robust Error Handling:** Safely detects and alerts users about empty or structurally corrupted files.
